```bash
npm install logmoji --save
```

Usage:

```javascript
const logger = require("logmoji")({ timestamp: false });

logger.success("You did it!");
// Output:  ✅ You did it
logger.fail("You're suck!");
// Output:  📛 You're suck
logger.error("Oh crap!");
// Output: 🚨 Oh crap!
```
