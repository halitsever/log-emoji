# Available config parameters:

timestamp: `true` | `false`<br>
logSymbols: `{
    success: string;
    fail: string;
    error: string;
    info: string;
    warn: string;
    log: string;
}`<br/>
disablePrefixText: `true` | `false`

# Disabling logs for prod:

add `LOG=false` to env file as env variable
